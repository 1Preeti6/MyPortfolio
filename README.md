- ⚡ **Single file, zero dependencies** — pure HTML/CSS/JS, no build step
- 🌙 Dark, techy aesthetic with animated grid, gradient mesh & cursor glow
- ⌨️ Typing animation, scroll-reveal sections, smooth micro-interactions
- 📄 Downloadable résumé
- 🔗 Clickable certification badges (AWS, Google GenAI, HackerRank)
- 📱 Fully responsive (desktop, tablet, mobile)
